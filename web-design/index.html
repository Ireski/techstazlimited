Course Overview

What You'll Learn

A complete roadmap from absolute beginner to professional WordPress designer. No prior experience or coding skills needed.

WordPress Foundations

Set up and navigate WordPress with confidence. Learn the dashboard, posts, pages, menus, and how the CMS is structured.

Themes & Design

Choose, install, and customise themes to match any brand. Master typography, colour palettes, and layout using the WordPress Customiser.

Module 2

Elementor Page Builder
